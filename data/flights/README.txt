This folder holds the raw data for the T-100 flight data. The data is downloaded from 
https://www.transtats.bts.gov/DL_SelectFields.asp?Table_ID=259&DB_Short_Name=Air%20Carriers
and 
https://www.transtats.bts.gov/DL_SelectFields.asp?Table_ID=258&DB_Short_Name=Air%20Carriers
for segment and market data, respectively, for all data fields for 2016.
